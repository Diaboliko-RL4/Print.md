# Print.md 🖨️

**We make prints everywhere!**

A professional printing services website showcasing our commitment to delivering high-quality printed materials to customers anywhere.

## 📋 Project Overview

Print.md is a modern, responsive website for a printing services company. It features:

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Professional Layout**: Clean and modern user interface
- **Services Showcase**: Display of various printing services offered
- **Contact Form**: Easy way for customers to reach out
- **Smooth Navigation**: Intuitive user experience with smooth scrolling

## 🎯 Features

- 📄 Document Printing
- 🎨 Custom Designs
- 📸 Photo Printing
- 🏷️ Labels & Stickers
- 📋 Banners & Posters
- 🎁 Gift Items

## 🚀 Getting Started

### Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/Diaboliko-RL4/Print.md.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Print.md
   ```

3. Open `index.html` in your web browser

### Deployment

This site is ready to be deployed on GitHub Pages:

1. Go to repository Settings
2. Navigate to "Pages"
3. Select "main" branch as source
4. Your site will be available at: `https://Diaboliko-RL4.github.io/Print.md/`

## 🎨 Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
--primary-color: #2c3e50;
--secondary-color: #e74c3c;
--accent-color: #3498db;
```

### Content
Update text and images directly in `index.html`

### Services
Add or modify service cards in the Services section of `index.html`
